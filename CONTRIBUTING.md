## Git Commit Message

- pattern: `type(scope): description`
- `type`: refer to [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/#summary)
- `scope`:
  - `api`: code changes in `./src/server/api`
  - `core`: code changes in `./src/server/core`
  - `client`: code changes in `./src/client`
