Acontext Community provides a collection of integrations for popular AI frameworks and tools.


## The easiest way to get started
Download `acontext-cli` and create a new project with the template:
```bash
curl -fsSL https://install.acontext.io | sh
acontext create my-proj --template-path "python/openai-agent-artifacts"
```


For more e2e integrations, please refer to the [Acontext-Examples](https://github.com/memodb-io/Acontext-Examples) repository.
